# Healthcare Analytics Portfolio

This repository contains Excel-based healthcare analytics projects.

## Project 01: Breast Cancer Analysis
**Objective:**  
Analyze breast cancer data to understand how tumor characteristics relate to diagnosis.

**Workflow:**
1. Raw data ingestion
2. Diagnosis converted to binary values
3. Average feature values calculated
4. Tumors categorized as Small / Medium / Large
5. Bar chart created for visual comparison

**Tools:**  
- Microsoft Excel

- ## Skills Demonstrated
- Healthcare data understanding
- Data cleaning and transformation
- Basic statistical analysis
- Data categorization
- Visualization using Excel
